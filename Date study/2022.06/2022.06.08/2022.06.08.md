### JSP (Java Server Page) : 템플릿 엔진
- 내부에서 자원을 갖고있다(HTML)
- Java Server Page라서 Java Pakage 안에 있어야 한다.
- HTML+CSS+Javascript (+Java)
- Ajax 사용이 가능하다!
- return type이 file 이름이다.
- JSP는 Project 안에 있어 내부에서 호출하여 자원으로 인식
- JSP를 외부로 빼면 실행이 되지 않는다.
- jsp -> jstl
- jstl을 이용해 java coding을 따로!
